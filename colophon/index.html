Colophon

I launched this website in 2009, initially as a personal blog, before later expanding it to include a professional portfolio. Throughout the years, I have explored different publishing platforms: from WordPress and OpenScholar to hand-written HTML. My journey led me to adopt the static website generator: blogdown, driven by Hugo.

This page lists the tools I use to build the site today. Besides the proprietary fonts, the entire stack is free and open source, and can be found on GitHub. The source code for this site itself is available at https://github.com/nanxstats/nanx.me.

Typography

  • Instrument Sans Modded
    Neo-grotesque typeface derived from Instrument Sans
  • Martina Plantijn
    Serif typeface designed by Kris Sowersby
  • Paper Mono Ligaturized
    Code font with Fira Code ligatures
  • Fira Sans Extra Condensed
    Favicon typeface

Publishing

  • blogdown
    Static site generator for R, driven by Hugo
  • hugo-nanx2026
    Responsive Hugo theme built on Tailwind CSS
  • utterances
    Lightweight commenting system using GitHub issues

Editing

  • Visual Studio Code
    Code editor from Microsoft
  • R extension
    R language support in VS Code
  • textmate.rstheme
    Light VS Code theme optimized for R and R Markdown
